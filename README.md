**Smart Product Recommendation System**

A simple web-based product recommendation system developed using Python and Flask.
The system allows users to search for a product (e.g., Mobile) and recommends related accessories such as charger, power bank, headphones, etc., with price and image.
This project is developed as a **Final Year Major Project**, inspired by e-commerce platforms like Flipkart.

Technologies Used
- Python
- Flask
- HTML, CSS

How to Run
1. Run `python app.py`
2. Open browser: http://127.0.0.1:5000

Output
Displays recommended products with name, price, and image.
